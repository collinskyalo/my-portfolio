# Collins Kyalo — Portfolio

This repository contains the static portfolio site for Collins Kyalo. The site is built with plain HTML, CSS, and JavaScript and is served as a static site.

Files:

- `index.html` — main HTML page
- `styles.css` — extracted CSS
- `script.js` — extracted JavaScript

Preview locally:

```powershell
python -m http.server 8000
# then open http://localhost:8000
```
